# 智能Mock系统产品文档

## 1. 产品定位

面向接口联调与测试场景，通过“接口文档自动解析 + Mock 数据生成”，快速构建可校验、可动态响应的 Mock 服务，提升研发联调效率。

## 2. 目标用户

- 后端开发
- 测试工程师
- 业务联调人员

## 3. 核心价值

- 文档即接口：上传文档即可生成 Mock
- 自动校验：必填参数校验，降低联调错误
- 动态响应：查询类接口响应随请求变化
- 统一管理：接口、文档、日志集中管理

## 4. 功能模块

### 4.1 首页

- 系统定位展示
- 快捷入口导航

### 4.2 接口文档管理

功能：

- 上传 Word / PDF 文档
- 查看原始解析结果
- 列表展示已上传文档，支持下载

### 4.3 接口管理

功能：

- 展示历史 Mock 接口列表
- 展示 Mock 地址与校验规则
- 一键跳转“在线调试”

### 4.4 在线调试

功能：

- 输入 Mock 地址
- 粘贴请求参数 JSON
- 立即获取响应结果

### 4.5 请求日志

功能：

- 操作日志记录（上传、命中、生成、错误、校验失败）
- 统计展示（累计、24小时、按接口维度）

## 5. 核心流程

1. 上传接口文档
2. 系统解析并生成 Mock
3. 复制 Mock 地址进入联调
4. 在线调试 / 真实调用
5. 日志与统计追踪

## 6. 关键业务规则

- 必填字段由模型识别并校验
- 错误 mock 通过特殊字段触发
- 动态响应对查询类接口生效

## 7. 成功指标

- 文档解析准确率
- Mock 生成时长
- 联调效率提升

## 8. 演示重点

建议演示流程：

1. 上传文档 → 自动生成 mock
2. 进入接口管理 → 展示 mock 列表
3. 在线调试 → 校验 + 动态响应
4. 请求日志 → 统计图表
